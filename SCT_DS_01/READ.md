 SkillCraft Internship Task 01

 Task Description
Create a histogram or bar chart to visualize the distribution of ages in a sample population using Microsoft Excel.

 Files Included
1. **Age_Distribution.xlsx** – Excel file containing the histogram.  
2. **Histogram_Screenshot.png** – Screenshot of the final histogram for professional presentation.

## Objective
- Understand data distribution using histograms.  
- Learn to create, format, and label Excel charts.  
- Prepare professional charts for reports or presentations.

## Tools Used
- Microsoft Excel 2016
